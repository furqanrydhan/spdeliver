## Installation

```bash
pip install spdeliver
```

or

```bash
pip install -e "git+http://github.com/stylepage/spdeliver.git#egg=spdeliver"
```

or

```bash
git clone git@github.com:stylepage/spdeliver.git spdeliver
pip install -e spdeliver
```
